# IOTDEMO
IOT demo files for MQTT presentation to 6th formers


requires some hardware, Wemos D1 was used for demo

and some IOT actuators, 
I used 3 Tasmota powewer devices
Plug:  re programmed Sonoff S20
Light: re programmed Sonoff basic
Dials: ESP-01, with Tasmota, generic config, andtwo PWM output
